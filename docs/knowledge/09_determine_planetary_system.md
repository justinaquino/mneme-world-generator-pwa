# Chapter 9 — Determine Planetary System

**Source:** https://wiki.gi7b.org/index.php/Mneme_World_Generator/Determine_Planetary_System
**Last Synced:** 2026-03-04

---

## Six-Step Process

1. Note solar system zone info, main world position, companion stars.
2. Roll per **Planetary Systems Table** for quantity of each body type. Roll 2D6 for each body's mass with star-type modifiers; optionally refine with Planetary Mass Modifier Table.
3. Use **Planetary System Qualities and Position Table** to assign zones; calculate exact distances using Main World Position Table. Handle orbital collisions by distributing positions between an orbit and 1.5× that orbit.
4. Determine **Circumstellar Disk zones** from Disk Zones Table; resolve collisions similarly.
5. Optionally generate asteroids within disks (Planetoid-Asteroid Mass Table + modifiers; -2 cumulative penalty for successive asteroids).
6. Optionally roll **Gas World Type Table** for detailed gas giant classifications.

---

## Body Types

- Circumstellar Disks
- Dwarf Planets
- Terrestrial Worlds
- Ice Worlds
- Gas Worlds

## Star-Type Modifiers to Mass Rolls

| Star Class | Modifier |
|------------|---------|
| O, B, A | Only Circumstellar Disks |
| F | Advantage +2 |
| G | Advantage +1 |
| K | No modifier |
| M | Disadvantage +1 |
