# Chapter 6 — Determine Habitability

**Source:** https://wiki.gi7b.org/index.php/Mneme_World_Generator/Determine_Habitability
**Last Synced:** 2026-03-04

---

## Overview

Each world after the main one receives **+1 Difficulty modifier** to habitability rolls.

### Sequential Rolls Required
1. Atmospheric Pressure
2. Temperature
3. Hazard Type (if any)
4. Hazard Intensity
5. Biochemical Resources
6. Population Modifiers (Primary and Secondary)

---

## Atmospheric Pressure

Roll **2D6**:

| Roll | Type | Habitability Modifier |
|------|------|-----------------------|
| <1 | Crushing | -2.5 |
| 2–5 | Dense | -2 |
| 6–8 | Trace | -1.5 |
| 9–11 | Thin | -1 |
| 12 | Average | 0 |

| Type | Description |
|------|-------------|
| **Average** | Earth-standard 0.5–3 atm (ideally 1 atm / 14.7 psi) |
| **Thin** | Weather exists; pressurized suits required (like Mars) |
| **Trace** | No weather; vacuum environmental suits required |
| **Dense** | TL10 suits required; 3–136 atm |
| **Crushing** | >136 atm; TL9+ buoyant habitats or specialized equipment |

## Temperature

Roll **2D6**:

| Roll | Type | Habitability Modifier |
|------|------|-----------------------|
| <2 | Inferno | -2 |
| 3–6 | Hot | -1.5 |
| 7–10 | Freezing | -1 |
| 11 | Cold | -0.5 |
| >12 | Average | 0 |

**Atmosphere modifiers to Temperature roll:**
- Dense atm: +1
- Crushing atm: +2
- Thin atm: -1
- Trace atm: -2

| Type | Description |
|------|-------------|
| **Average** | -25°C to 45°C (avg 16°C) |
| **Cold** | -60°C to -140°C; TL6 required |
| **Hot** | 30°C to 100°C; between habitable and hot zones |
| **Freezing** | No significant heat source; artificial systems needed |
| **Inferno** | >100°C/212°F; often stripped of atmosphere |
