## Advantage / Disadvantage System

Notation: **Adv+(number)** or **Dis+(number)**

Advantage and Disadvantage cancel each other:
- Adv+2 + Dis+1 = Adv+1

### Rolling with Advantage

Roll additional dice equal to the advantage value, then drop the lowest results.

> "Adv+2 to a 1D6 roll means roll 3D6 but drop the 2 dice with the lowest result."

### Rolling with Disadvantage

Same process but drop the highest results.

> "Dis+3 to a 2D6 roll means roll 5D6 but drop the 3 dice with the highest result."
