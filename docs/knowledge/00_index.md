## Overview

The **Mneme World Generator** is a set of variant rules for the Cepheus Engine tabletop RPG system. It enables detailed generation of solar systems, worlds, inhabitants, and societies for science fiction settings, derived from the Traveller System Reference Document and Open Gaming Content.

**Authors:** Justin Cesar Aquino, Niccolo Salonga
**Proofreading/Art:** Niccolo Salonga
**Spreadsheet Contributors:** John Karol Abayan, Lloyd Reyes, Ranz Sapinit, Michael Ibanez, Christine Melody Taa
